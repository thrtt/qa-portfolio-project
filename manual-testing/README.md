# Manual Testing - SauceDemo

This section contains manual testing documentation for the SauceDemo web application.

## Application Under Test

SauceDemo is a demo e-commerce web application used for practicing software testing.

URL: https://www.saucedemo.com/

## Scope of Testing

The following functionalities are covered:

- Login
- Product inventory
- Add to cart
- Cart page
- Checkout process
- Logout

## Test Documentation

This folder includes:

- Test Plan
- Test Scenarios
- Test Cases
- Bug Reports
- Test Summary Report

## Testing Approach

Manual functional testing was performed to verify that the core features of the application work as expected. Both positive and negative test scenarios are included.
