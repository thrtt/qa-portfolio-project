### CTC-001 - Verify one added product is displayed in cart

**Scenario:** CS-001

**Preconditions:**
- User is logged in with valid credentials
- User is on the inventory page

**Steps:**
1. Add one product to the cart
2. Click on the cart icon
3. Verify one added product is displayed in the cart

### CTC-002 - Verify multiple added products are displayed in cart

### CTC-003 - Verify user can remove one product from the cart

**Scenario:** CS-002

**Preconditions:**
- User is logged in with valid credentials

**Steps:**

4. Verify the removed product is no longer displayed in the cart

### Verify user can continue shopping from cart

**Scenario:** CS-003

**Preconditions:**
- User is logged in with valid credentials
- User has added at least one product to the cart
- User is on the cart page

**Steps:**
1. Click "Coninue Shopping" button
2. Verify the user is redirected to the inventory page

### Verify user can proceed to checkout

**Scenario:** CS-004

**Preconditions:**
- User is logged in with valid credentials
- User has added at least one product to the cart
- User is on the cart page

**Steps:**
1. Click the "Checkout" button
2. Fill the fields "First Name", "Last Name", and "Postal Code"
3. Click the "Continue" button
4. Click the "Finish" button
5. Verify the message "Thank you for your order!" is displayed
