# Improvement Suggestions - SauceDemo

---

### OBS-001 - Short automatic logout timeout

The application logs out the user after a short period of inactivity.

Suggestion:
Consider increasing the session timeout duration to improve user experience.

---

### OBS-002 - Checkout flow does not include a payment step

The checkout process can be completed without selecting or entering any payment method.

For a demo e-commerce application, a simulated payment step could make the checkout flow more realistic and closer to a real user experience.

Suggestion:
Add a mock payment method step, such as selecting "Demo Card" or entering fake card details, before completing the order.
