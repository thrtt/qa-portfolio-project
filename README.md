# QA Portfolio Project

This project demonstrates my QA skills in manual testing, API testing, and UI automation.
This project is part of my preparation for a junior QA position.

## Project Overview

The goal of this project is to simulate a real QA workflow by:
- Creating test cases
- Identifying potential defects
- Testing API endpoints
- Writing basic UI automation tests

## Scope

- Manual testing of core functionality
- Example bug report
- API testing using Postman
- UI automation using Playwright and Selenium

## Tools Used

- Playwright
- Selenium WebDriver
- Postman
## What I Learned

- How to structure test cases
- How to analyze application behavior
- How to validate API responses
- Basic automation concepts

## Future Improvements

- Add more test scenarios
- Expand automation coverage
